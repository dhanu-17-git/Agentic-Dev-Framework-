# Full Feature Pipeline

## Flow
planner → coder → tester → reviewer → security → documenter

## Steps
1. **Planner** — breaks feature into tasks with file references
2. **Coder** — implements each task with minimal edits
3. **Tester** — generates unit and integration tests
4. **Reviewer** — audits code quality and architecture
5. **Security** — scans for vulnerabilities
6. **Documenter** — updates architecture docs and README

## Exit Criteria
- All tests pass
- Reviewer approved
- Security check cleared
- Docs updated
