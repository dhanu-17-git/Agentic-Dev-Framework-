# Decision Log

> **System:** Project Memory  
> **Updated by:** After every brainstorming session and major design choice

## Purpose
Record major design decisions so AI and humans can trace why choices were made.

## Decision Template

### Decision #[N]: [Title]

**Date:** YYYY-MM-DD  
**Context:** [What problem or feature triggered this decision]  
**Decision:** [What was decided]  
**Alternatives Considered:**
- [Option A] — rejected because [reason]
- [Option B] — rejected because [reason]

**Impact:** [What this affects — files, modules, performance, users]  
**Status:** Accepted / Superseded by Decision #[X]
